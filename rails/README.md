# expo_turbo-rails

The Rails package for Expo Turbo.

This directory currently contains only a buildable, non-route-owning Rails Engine scaffold. It does not register MIME types, renderers, helpers, jobs, channels, or routes and does not claim Turbo compatibility yet.

```ruby
gem "expo_turbo-rails"
```

```ruby
require "expo_turbo/rails"
```

See the repository [README](../README.md) for project status and development commands.
